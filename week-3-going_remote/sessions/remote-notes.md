Explain:
● What is a remote repository?
● What is origin?
● Why do developers use GitHub?
● Difference between local and remote repositories.


#ANSWERS
- A remote repository is a repository that lives on a cloud-based hosting platform like GitHub.
It is a copy of the repo on your local machine.

- Origin refers to the shortform or a representation of your repo's URL.

- Developers use GitHub because it serves as a resume. It keeps a copy their projects so they don't
get lost when say, the developer's machine crashes. GitHub also allows different developers to 
contribute on a project easily.

- A local repository is the one that stays on your local machine while a remote repository is a 
version of your local repository saved in the cloud.
