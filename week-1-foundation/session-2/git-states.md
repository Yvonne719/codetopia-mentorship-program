#The Three(3) Git States

Project files, before they are permanently stored in a version history go through some states. These will be listed here.

* Working Directory - This location holds files that are to be edited before being moved to their permanent locations.

* Staging Area - This location is like a preparatory area for project files that are to be committed into a repository.

* Repository - This is the permanent location for files that have been edited and staged.
